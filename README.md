# BetterDiscord Video Resizer

Resize video embeds in Discord chat by dragging the bottom-right corner handle.

Supports YouTube, Twitter/X, uploaded videos, and more.

---

## Features

- Drag-to-resize video embeds directly in chat
- Works on common embed types (YouTube, Twitter/X, direct uploads, and more)
- Simple, lightweight, and easy to use

---

## Usage

1. Find a message containing a video embed.
2. Hover the embed (if needed).
3. Drag the **bottom-right corner handle** to resize the video.
